Projects

HTML5 jQuery Drag Drop Interactive UI

This is a fun job to be implemented in HTML5. (see attached illustration)

Part of a larger project this will be later integrated. I can provide the classModels but you will need to create the MVC4 backend just to support this UI.

The UI consists of a list of drivers down the left side. The right side has a list of jobs. The remainder of the screen is for the scheduling to drivers.
A selection of drivers can be made. Selecting multiple drivers means a number of columns appear. Scheduled jobs are shown for the driver. Dragging a new job from the pool on the right side of the screen and dropping it on a driver will schedule the job. You will see in the diagram that the scheduled jobs should separate to illustrate where the new job would be inserted in the schedule (should the user release/drop).

Scheduling is for a given date - so the whole screen is date sensitive. The drop down calendar should be easy to navigate. The arrows indicate forward and back date
